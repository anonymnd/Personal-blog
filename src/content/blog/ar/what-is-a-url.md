**عنوان URL** (*Uniform Resource Locator*) هو العنوان الذي تكتبه أو تنقره للوصول إلى مورد على الإنترنت—غالباً صفحة ويب أو واجهة برمجية (API).

## مثال

```
https://example.com/docs/search?q=astro&lang=en
```

## الأجزاء الرئيسية

| الجزء        | في المثال          | الدور                                                         |
| ------------ | ------------------ | ------------------------------------------------------------- |
| **المخطط**   | `https://`         | يحدد *كيف* يتحدث المتصفح مع الخادم (هنا HTTP مشفّر).          |
| **المضيف** | `example.com`      | يحدد الخادم (عبر DNS إلى عنوان IP).                           |
| **المسار**   | `/docs/search`     | أي مستند أو مسار تريده على ذلك الخادم.                        |
| **الاستعلام** | `?q=astro&lang=en` | معاملات إضافية، غالباً من نماذج أو مرشحات.                    |

لا حاجة لحفظ كل التفاصيل في البداية؛ يكفي أن تميّز هذه القطع عند قراءة الوثائق أو ضبط الاستضافة.
